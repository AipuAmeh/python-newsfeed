# Pyython-Newsfeed

## Description

NewsFeed is a refactored tech blog that uses python for its backend technology instead of node.

## Installation

The technologies used in this app are:
- Python3
- SQLAlchemy database
- Flask
- Jinja Templates
- Procfile
